# @dbsketch/parser

Internal parser package for [dbsketch](https://github.com/jacobmpeters/dbsketch). Parses DBML (with the `@layout` extension) and SQL DDL into a normalized IR.

Most users want [`@dbsketch/core`](https://www.npmjs.com/package/@dbsketch/core) (library) or [`@dbsketch/cli`](https://www.npmjs.com/package/@dbsketch/cli) (CLI) instead.
