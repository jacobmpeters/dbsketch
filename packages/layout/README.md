# @dbsketch/layout

Internal layout package for [dbsketch](https://github.com/jacobmpeters/dbsketch). Grid-native deterministic placement and edge routing on a strip-grid model — integer cells from start to finish, no continuous-space projection.

Most users want [`@dbsketch/core`](https://www.npmjs.com/package/@dbsketch/core) (library) or [`@dbsketch/cli`](https://www.npmjs.com/package/@dbsketch/cli) (CLI) instead.
