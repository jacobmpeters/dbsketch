# @dbsketch/render

Internal rendering package for [dbsketch](https://github.com/jacobmpeters/dbsketch). Turns a strip-grid layout into a string using either an ASCII or Unicode box-drawing glyph table.

Most users want [`@dbsketch/core`](https://www.npmjs.com/package/@dbsketch/core) (library) or [`@dbsketch/cli`](https://www.npmjs.com/package/@dbsketch/cli) (CLI) instead.
